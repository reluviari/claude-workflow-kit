# Bugfix Workflow

Use this workflow for bugs and regressions.

## Steps

1. Reproduce or describe the bug.
2. Identify expected behavior.
3. Identify actual behavior.
4. Inspect related code and tests.
5. Find likely root cause.
6. Propose minimal fix.
7. Add regression test when possible.
8. Implement fix.
9. Run checks.
10. Summarize cause and fix.

## Output

- bug summary
- root cause
- fix
- regression test
- commands executed
- risks
